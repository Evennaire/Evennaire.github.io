| ≥100                |               |
| ------------------- | ------------- |
| (ein)hundert        | 一百100       |
| (ein)tausend        | 一千1,000     |
| (ein)hunderttausend | 十万100,000   |
| eine Million        | 百万1,000,000 |

1. 两位数，个位+und+十位，如76-sechsundsiebzig
2. 百万以下，全部是一个单词，如**abc,def **- **a** hundert **c** und **b** tausend **d** hundert **f** und **e**，注意hundert后面不加und，und只有在几十几的时候才加

3. 百万以上，先说多少Millionen，再说后面百万以下的

如2,345,678：

zwei Millionen dreihundertfünfundvierzigtausendsechshundertachtundsiebzig

4. 小数点读作komma，小数点后数字一位一位读，如38.51：achtunddreßig komma fünf eins
