生成公钥私钥

<code>ssh-keygen -t rsa</code>

查看公钥

<code>cat ~/.ssh/id_rsa.pub</code>

查看某次修改，--stat表示只看文件名

<code>git show [commit ID] --stat</code>

查看两次commit之间的修改

<code>git diff [commit ID 1] [commit ID 2] --stat</code>
