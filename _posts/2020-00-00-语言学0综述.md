语料库是能力的真子集，即语料库不能穷尽所有的句子，你一定可以说出不在语料库里的句子。

语言学最核心的五个领域：

+ 语音学Phonetics
  + 人类语音的基本单位
  + 如何描写和记录这些单位
  + 单位在发音声学听觉上的特征
+ 音系学
  + 语音的基本单位如何组合
  + ？
+ 形态学：研究“词”的内部构造
  + 每种自然语言的“词”的表现形式
  + 词的内部结构
  + 词内组成成分的类别
+ 句法学：研究词组和句子内部构造
  + 词是如何构成词组和句子的
  + 句子之间在结构上的派生关系
  + 例如：英语问句的语法
+ 语义学 Sense
  + 词的意义的分析
  + 词组和句子的意义的得来

另外还有一个领域：

+ 语用学：
  + 如何使用语言来达到交际目的
  + 语境如何赋予语言单位所没有的字面意义

交叉学科：

+ 语言社会学Sociology
+ 语言哲学Philosophy
+ NLP
+ 心理语言学：儿童如何学会说话？





law（法）的两种含义：

+ 客观存在的规律
+ 人们根据客观规律制定的规则

语法的两种含义：

+ 自然语言在组织和构造等方面的客观规律：描写性语法
+ 语言学家根据规律制定出来的语法：规定性语法
