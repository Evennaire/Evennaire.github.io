# 语音学Phonetics

## 基础

### 语音的两种理解

+ 狭义：人类发声器官发出的、有声语言传递信息的物质载体
+ 广义：还包括用来作为手语传递信息的物质载体，即手势等肢体动作

### 语音学的分支

1. 发声语音学(articulatory phonetics)：解剖生理的角度
2. 声学语音学(acoustic phonetics)：物理的角度
3. 听觉语音学(auditory phonetics)：听觉（神经心理）的角度
4. 实验语音学(experimental phonetics)

### 发音原理

+ 动力源
  + 呼气音：以肺部呼出的气流作为发声动力
  + 吸气音：以口腔吸入的气流作为发生动力
+ 音响源
  + 声带：气流通过声门影响声带，声带震动与否决定所发音的响度
+ 音色源
  + 发声通道：气流在发生通道的某一部位受到调整或阻塞，然后通过口腔或鼻腔进行释放。气流被调整或阻塞的部位和方式以及释放的途径决定语音的音色

### 发音过程

1. 气流产生过程airstream process：肺部呼出气流
2. 发声过程phonation process：气流经过声门，影响声带
3. 口鼻过程oral-nasal process：气流继续往上，经过口腔或鼻腔
4. 成音过程articulatory process：气流在释放前在声道某处得到调整或阻塞

### 语音形成的四个重要参数

1. 呼气音Pulmonic or吸气音Non-pulmonic 

   大多数语言中的大多数语音都是呼气音（由肺部呼出气流作为发声动力）

   少数语言，主要是非洲的一些语言，使用吸气音（吸入空气作为动力源）

   现代汉语里唯一一个吸气音：啧（舌尖吸气音）

2. **清浊Voicing：声带是否闭合**

   - 浊音Voiced：声带**闭合**，声音通过会使声带**震动**，**响度高**，包括**所有元音**和部分辅音

   - 清音Voiceless：声带打开，声音通过不会震动声带，响度低，一般都是辅音

3. 口音or鼻音Nasality：软腭，潜意识控制

   - 口音oral：软腭顶起
   - 鼻音nasal：软腭下垂

4. 音色：多值参数

   元音和辅音的分野：

   - 辅音：气流在发生通道的某处受到某种程度的**阻塞**，被阻塞的部位和程度方式决定音色
   - 元音：气流在发生通道里受到调整但**不受到阻塞**，舌头、口腔、嘴唇等决定音色

### 国际音标International Phonetic Alphabet (IPA)

最新的国际音标（2015）包括：

+ 107个字母letters

+ 31个修音符号diacritics：对语音进行特征调整

+ 19个附加符号additional signs：用于描写附着于语音之上的超音段特征

用方括号[]将发音围起来

指导原则：

+ 一音一符，一一对应
+ 以自然语言为准，两个不同的音，如果没有任何一种自然语言区别它们，一般不设置两个符号来代表

## 辅音Consonant

（只关注呼气辅音）

### 决定辅音音值的四个参数

+ 清or浊：声带是否闭合，辅音有清有浊
+ 口音or鼻音：软腭状态，
+ 发音部位（阻塞的部位）
+ 发音方式（阻塞的具体程度和方式）

后两个是多值参数。

描写方式：发音部位-清浊-口鼻-发音方式

某个参数的说明可以省略，如：

- 双唇清(口)塞音[p]

- 舌尖浊(鼻)塞音[n]

### 发音部位

主动发音器官（可以移动的部位）和被动发音器官（不能移动的部位），发音部位是主动器官与被动器官**接触或接近**的部位

+ 双唇bilabial
+ 唇齿labiodental，一般是上唇+下齿，如[f]、[v]
+ 齿间interdental，现代汉语没有，英语[θ]、[ð]
+ 舌尖前alveolar，舌尖接近或接触齿龈产生阻塞[d]、**[s]**、[l]
+ 舌尖后palatao-alveoalr，舌尖后部接近或接触硬腭前，[ʃ] [ʒ] [t͡ʃ] [d͡ʒ]，舌头是平的（与卷舌音不同）
+ 舌面palatal，舌身接近或接触硬腭，[j] [c]
+ 舌根velar，舌根接近或接触软腭，[g] [k] [x]

汉语里特殊的：

+ 卷舌音，与舌尖后音相似但有不同：卷舌音是卷着舌尖接触齿龈后部的。**国际音标里把卷舌作为一种发音部位**

  like, shoe 和“书”

以下汉语里没有，印欧语里出现：

+ 小舌音uvular
+ 喉音pharyngeal，咽部的喉壁为发音部位
+ 声门音glottal，声门（喉结里的声带区）

### 发音方法

常见的辅音发音方法：

+ 塞Stop：完全阻塞
  + 口塞音oral stop：爆破音explosive，不可延续
  + 鼻塞音nasal stop：**都是浊音**（因为如果是清音声带打开，气流从鼻腔出则没法发音）
  + 所以可以省略：清(口)塞音、浊(鼻)塞音
+ 擦Fricative：气流在细缝中绵延摩擦而出
  + 可以在发音上延续，如[f] [v]
  + 都是**口音**，没有鼻音，口擦音（鼻怎么擦？）
  + 清[ʂ]，浊[ʐ]
+ 塞擦Affricate：先塞后擦，实际发音中两个过程浑然一体
  + 如：在zai的声母 [t͡s]
  + 应该也是结合了塞和擦的特点，都是口塞擦音
+ 边Lateral：中间塞，两边（或一边）通气，气流**实际**没有得到堵塞，响度较大
  + 如：[l]
  + 都是**浊音**
+ 近/通Approximant：有较大通气通道，气流没有堵塞（辅音和元音的灰色地带）
  + yes-[y]
+ 颤音Trill
+ 闪音Tap or Flap

另一个描写辅音的参数：

+ 送气(aspiration)：与发音开始时间有关。
  + 只涉及到塞音和塞擦音，发声语音学用四个参数无法刻画，只能用声学“是否送气”来刻画
  + 如汉语中b和p，d和t，zh和ch的区别
  + 用上标^h^来表示是否送气

### 英汉比较

+ 清浊：英语的浊音更多
  + 英语中的口塞音、擦音、塞擦音都有清浊对立
  + 汉语只有四个浊音：[m] [n] [l] [r]
+ 发音部位：英语的发声通道前后部更多，汉语中间部较多
  + 英语没有卷舌音
  + 汉语没有舌尖后
  + 英语的舌面只有一个近音，汉语有三个舌面音jqx（龈-腭音）
+ 发音方法：汉语的塞擦音更多

## 元音

### 元音和辅音的发声差异

1. 元音是发声通道没有阻塞所发出来的语音
2. 元音产生时声带发生震动，所以**元音都是浊音**，而辅音有清有浊

### 决定元音音色的三个参数

+ 舌位的前后 ()
+ 舌位的高低 (tougue height)
  + 指舌头离上颚的距离
+ 嘴唇的圆窄

其它影响元音音色的因素：

+ 肌肉的松紧
  + 指发音器官的松紧（主要是舌头）会改变作为共振腔的口腔的形状，从而影响音色
  + 紧元音or松元音
  + 如英语中的长元音和短元音，sheet vs shit，beach vs bitch
+ 卷舌动作
  + 每个元音都可以有一个与之配对的卷舌元音

## 超音段

### 总述

音段：单个辅音/元音或辅音+元音组合起来的更大的语音片段

超音段：覆盖在音段之上的声学特征

被人类语言选用来作为超音段成分的三个声学特征：

+ 音长duration
+ 音强loudness
  + 体现为轻重音
  + 使用
    + 英语中：多个音节的轻重分配，国际音标用[]
    + 汉语中：较少，如东西vs东西（轻）意义不同，大多数轻重音意义相同；不是强制的
    + 句子层面，用重音来标注语法信息，比如对比焦点
  + 
+ 音高pitch
  + 语调intonation
    + 依附在句子上
    + 传递不同的句子上的语法意义：语气，如陈述、疑问、祈使、感叹等
    + 是普遍的，所有的人类语言都有
  + 声调tone
    + 依附在音节上的
    + 传递不同的词汇意义
    + 部分语言有，如汉语
  + 语调和声调可以共存

### 声调

+ 调型
  + 平调level tone：音高从头至尾不发生变化
  + 升调rising tone：由低到高
  + 降调falling tone：由高到低
  + 曲折调contour tone：有高低的曲折变化
+ 调值
  + 实际音高频率（及其变化），但不同人说的频率不同。母语者忽略了这种声学上调值的个体差异
  + 调类：根据是否影响词义将调值分类，如普通话有四个调类
  + 五度分类法（赵元任）：1-5，低-半低-中-半高-高
  + 不用给出具体的值，人与人不同，实测应该在一个范围内
+ 调域：用来描述调型音高变化的音高范围domain
+ 数字法：
  + 平调：33
  + 升调：35
  + 降调：31
  + 曲折调：351
  + 普通话：55,35,214,51
+ 符号法/图形法：画图

### 现代汉语的声调系统

+ 第一声，高平调，55，阴平
+ 第二声，高升调：35，阳平
+ 第三声，曲折调，214，上（shǎng）声
+ 第四声，降调，51，去声

最晚到南北朝时期，汉语肯定有声调了。南北朝时期的“四声”：平、上、去、入

清声母：阴声调

浊声母：阳声调

一共八类。

现代各方言都是从这八类演化而来的，入逐渐被合并到其它三个声调里。
