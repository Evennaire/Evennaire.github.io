# 语音学Phonetics

## 基础

语音的两种理解：

+ 狭义：人类发声器官发出的、有声语言传递信息的物质载体
+ 广义：还包括用来作为手语传递信息的物质载体，即手势等肢体动作



1. 发声语音学(articulatory phonetics)：解剖生理的角度
2. 声学语音学(acoustic phonetics)：物理的角度
3. 听觉语音学(auditory phonetics)：听觉（神经心理）的角度
4. 实验语音学(experimental phonetics)



+ 动力源
  + 呼气音
  + 吸气音
+ 音响源
  + 声带：气流通过声门影响声带
+ 音色源
  + 发声通道：气流在发生通道的某一部位受到影响，然后通过口腔或鼻腔进行释放

## 语音形成的四个重要参数

1. 呼气音Pulmonic or吸气音Non-pulmonic 

   大多数语言中的大多数语音都是呼气音（由肺部呼出气流作为发声动力）

   少数语言，主要是非洲的一些语言，使用吸气音（吸入空气作为动力源）

   现代汉语里唯一一个吸气音：啧（舌尖吸气音）

2. 清浊(Voicing)：声带

   - 清音Voiceless：声带打开，声音通过不会振动声带（一些辅音）
   - 浊音Voiced：声带闭合，声音通过会使声带振动（比较响，“啊”）

3. 口音or鼻音(Nasality)：软腭，潜意识控制

   - 口音Oral
   - 鼻音nasal

4. 音色

   元音和辅音的分野：

   - 辅音：气流在发生通道的某处受到某种程度的阻塞，被阻塞的部位和程度方式决定音色
   - 元音：气流在发生通道里受到调整但不受到阻塞，舌头、口腔、嘴唇等决定音色？

## 国际音标International Phonetic Alphabet (IPA)

最新的国际音标（2015）包括：

+ 107个字母letters

+ 31个修音符号diacritics：对语音进行特征调整

+ 19个附加符号additional signs：用于描写附着于语音之上的超音段特征

用方括号[]将发音围起来

指导原则：

+ 一音一符，一一对应
+ 以自然语言为准，两个不同的音，如果没有任何一种自然语言区别它们，一般不设置两个符号来代表

shoe-[ʃu]

sing-[θiŋ]

thank-[θæŋk]

### 发音部位

主动发音器官（可以移动的部位）和被动发音器官（不能移动的部位），发音部位是主动器官与被动器官**接触或接近**的部位

+ 双唇bilabial
+ 唇齿labiodental，一般是上唇+下齿，如[f]、[v]
+ 齿间interdental，现代汉语没有，英语[θ]、[ð]
+ 舌尖前alveolar，舌尖接近或接触齿龈产生阻塞[d]、[s]、[l]
+ 舌尖后palatao-alveoalr，舌尖后部[ʃ]，舌头是平的（与卷舌音不同）
+ 舌面palatal，
+ 舌根velar，[g] [k] [h]

汉语里特殊的：

+ 卷舌音，与舌尖后音相似但有不同：卷舌音是卷着舌尖接触齿龈后部的。**国际音标里把卷舌作为一种发音部位**

  like, shoe 和“书”

以下汉语里没有，印欧语里出现：

+ 小舌音uvular
+ 喉音pharyngeal，咽部的喉壁为发音部位
+ 声门音glottal，声门（喉结里的声带区）

<br>

### 发音方法

常见的四种辅音发音方法：

+ 塞：完全阻塞
  + 口塞音（爆破音explosive）
  + 鼻塞音：**都是浊音**
+ 擦Fricative：摩擦
+ 塞擦Lateral fricative：先塞后擦，实际发音中两个过程浑然一体
  + 如：在zai的声母 [t͡s]
+ 边：中间塞，两边通气
  + 如：[l]
+ 近/通(approximant)：有较大通气通道（辅音和元音的灰色地带；辅音和元音的区别：是否阻塞）yes-[y]
+ 颤音Trill
+ 闪音Tap or Flap

另一个描写辅音的参数：

+ 送气(aspiration)，但只涉及到塞音和塞擦音。发声语音学用四个参数无法刻画，只能用声学“是否送气”来刻画
  + 如汉语中b和p，d和t，zh和ch的区别
  + 发音开始时间。是一种语言内相对的发音开始时间
  + 用上标h来表示是否送气

## 元音

### 元音和辅音的发声差异

1. 元音是发声通道没有阻塞所发出来的语音
2. 元音产生时声带发生震动，所以**元音都是浊音**，而辅音有清有浊

### 决定元音音色的三个参数

+ 舌位的前后 ()
+ 舌位的高低 (tougue height)
  + 指舌头离上颚的距离
+ 嘴唇的圆窄

其它影响元音音色的因素：

+ 肌肉的松紧
  + 指发音器官的松紧（主要是舌头）会改变作为共振腔的口腔的形状，从而影响音色
  + 紧元音or松元音
  + 如英语中的长元音和短元音，sheet vs shit，beach vs bitch
+ 卷舌动作
  + 每个元音都可以有一个与之配对的卷舌元音
