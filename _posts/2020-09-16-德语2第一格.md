规定记号：

()括号内表示相对等的英文

第一格：N

## 冠词第一格

|                 | m    | f    | n    | Pl.  |
| --------------- | ---- | ---- | ---- | ---- |
| 定冠词(the)     | der  | die  | das  | die  |
| 不定冠词(a, an) | ein  | eine | ein  | /    |

## 名词第一格

单数和复数形式不同，一般单词表中会标出来。

e. g.

+ der Tisch, die Tische

+ die Frau, die Frauen
+ die Haus, die Häuser

## 人称代词第一格

|       | 我   | 你   | 他        | 您   |
| ----- | ---- | ---- | --------- | ---- |
| Sing. | ich  | du   | er/sie/es | Sie  |
| Pl.   | wir  | ihr  | sie       | Sie  |

## 动词现在时变位

|                 | sein (be) | haben (have) | heißen 名叫 | machen (make) | frag (ask) |
| --------------- | --------- | ------------ | ----------- | ------------- | ---------- |
| ich 我          | bin       | habe         | heiße       | mache         | frage      |
| du 你           | bist      |              | heißt       | machst        | fragst     |
| er/sie/es 他    | ist       |              | heißt       | macht         | fragt      |
| wir 我们        | sind      |              | heißen      | machen        | fragen     |
| ihr 你们        | seid      |              | heißt       | macht         | fragt      |
| sie/Sie 他们/您 | sind      |              | heißen      | machen        | fragen     |

大概记住：我+e，你+st，他/你们+t，我们/他们/您+en
