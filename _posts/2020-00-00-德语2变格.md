## 总览

| 释义                     | 记号           |
| ------------------------ | -------------- |
| 第一格（主格）           | Nominativ (N.) |
| 第二格（所有格）         | Genitiv (G.)   |
| 第三格（与格）当间接宾语 | Dativ (D.)     |
| 第四格（宾格）           | Akkusativ (A.) |

+ 第一格主要用作主语和属类补足语
  + <u>Der Student</u> lernt Deutsch.
  + Sie ist <u>Lehrerin</u>.
  + Das ist <u>ein Buch</u>.
+ 第二格主要用作定语，表示所属关系
  + die HauptStadt <u>der Provinz</u>
+ 第三格主要用作间接宾语，或接在支配第三格的介词后
  + 及物动词
    + Der Lehrer erklärt <u>dem Schüler</u> den Text.
  + 不及物动词
    + Hans hilft oft <u>seinem Freund</u>.
    + Ich danke <u>meinem Lehrer</u> für seine Erklärung.
  + 介词
    + Peter geht <u>zum Unterricht</u>.
    + Monika wohnt <u>bei ihren Eltern</u>.
+ 第四格主要用作直接宾语

<br>

## 冠词/物主代词变格

根据被修饰的名词的性，和整个短语在句子中所处的格来进行变化。

| 定冠词(the)<br/>不定冠词(a, an)<br/>物主代词(my) | m               | f               | n               | Pl.             |
| --------------------------- | --------------- | --------------- | --------------- | --------------- |
| Nominativ                   | der<br/>ein<br/>mein | die<br/>eine<br/>meine | das<br/>ein<br/>mein | die<br/>/<br/>meine |
| Akkusativ            | den<br/>einen<br/>meinen | die<br/>eine<br/>meine | das<br/>ein<br/>mein | die<br/>/<br/>meine |
| Dativ                       | dem<br/>einem<br/>meinem | der<br/>einer<br/>meiner | dem<br/>einem<br/>meinem | den<br/>/<br/>meinen |
| Genitiv            | des<br/>eines<br/>/      | der<br/>einer<br/>/      | des<br/>eines<br/>/      | der/<br/>/<br/>/     |

## 动词现在时变位

+ 规则动词：我+e，你+st，他/你们+t，我们/他们/您+en

    |                 | sein (be) | heißen 名叫 | machen (make) | frag (ask) | haben (have) |
    | --------------- | --------- | ----------- | ------------- | ---------- | ------------ |
    | ich 我          | bin       | heiße       | mache         | frage      | habe         |
    | du 你           | bist      | heißt       | machst        | fragst     | hast         |
    | er/sie/es 他    | ist       | heißt       | macht         | fragt      | hat          |
    | wir 我们        | sind      | heißen      | machen        | fragen     | haben        |
    | ihr 你们        | seid      | heißt       | macht         | fragt      | **habt**     |
    | sie/Sie 他们/您 | sind      | heißen      | machen        | fragen     | haben        |

+ 不规则动词-换音动词：

    |                 | sein (be) |      |      |      | haben (have) | geben (give) | essen (eat) |
    | --------------- | --------- | ---- | ---- | ---- | ------------ | ------------ | ----------- |
    | ich 我          | bin       |      |      |      | habe         | gebe         | esse        |
    | du 你           | bist      |      |      |      | hast         | gibst        | isst        |
    | er/sie/es 他    | ist       |      |      |      | hat          | gibt         | isst        |
    | wir 我们        | sind      |      |      |      | haben        | geben        | essen       |
    | ihr 你们        | seid      |      |      |      | **habt**     | **gebt**     | **esst**    |
    | sie/Sie 他们/您 | sind      |      |      |      | haben        | geben        | essen       |
