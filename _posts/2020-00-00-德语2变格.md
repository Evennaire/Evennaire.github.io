## 总览

| 释义                     | 记号           |
| ------------------------ | -------------- |
| 第一格（主格）           | Nominativ (N.) |
| 第二格（所有格）         | Genitiv (G.)   |
| 第三格（与格）当间接宾语 | Dativ (D.)     |
| 第四格（宾格）           | Akkusativ (A.) |

+ 第一格主要做名词，和属类补足语，即：
  + Sie ist Lehrerin.
  + Das ist ein Buch.
+ 第二格
+ 第三格主要做间接宾语，或接在支配第三格的介词后
  + 及物动词
    + Der Lehrer erklärt **dem Schüler** den Text.
  + 不及物动词
    + Hans hilft oft **seinem Freund**.
    + Ich danke **meinem Lehrer** für seine Erklärung.
  + 介词
    + Peter geht **zum Unterricht**.
    + Monika wohnt **bei ihren Eltern**.
+ 第四格主要做直接宾语

<br>

## 冠词变格

参数：性/数、格

| 定冠词(the)/<br>不定冠词(a, an) | m               | f               | n               | Pl.             |
| --------------------------- | --------------- | --------------- | --------------- | --------------- |
| Nominativ                   | der/ein Tisch   | die/eine Frau   | das/ein Haus    | die Tische/Tanten/Kinder |
| Genitiv                     |                 |                 |                 |                 |
| Dativ                       | dem/einem Tisch | der/einer Tante | dem/einem Kind | den Tischen/Tanten/Kindern |
| Akkusativ                   | den/einen Tisch<br> | die/eine Frau   | das/ein Haus   | die Tische/Tanten/Kinder   |

**注意：**

Dativ第三格：弱变化名词加-n/-en，复数时都加-n，除了以s结尾的外来词

## 名词变格

单数和复数形式不同，一般单词表中会标出来。

e.g.

+ der Tisch, die Tische
+ die Frau, die Frauen
+ die Haus, die Häuser

分为强变化/弱变化，变音/换音，留待整理。

## 动词现在时变位

+ 规则动词：我+e，你+st，他/你们+t，我们/他们/您+en

    |                 | sein (be) | heißen 名叫 | machen (make) | frag (ask) | haben (have) |
    | --------------- | --------- | ----------- | ------------- | ---------- | ------------ |
    | ich 我          | bin       | heiße       | mache         | frage      | habe         |
    | du 你           | bist      | heißt       | machst        | fragst     | hast         |
    | er/sie/es 他    | ist       | heißt       | macht         | fragt      | hat          |
    | wir 我们        | sind      | heißen      | machen        | fragen     | haben        |
    | ihr 你们        | seid      | heißt       | macht         | fragt      | **habt**     |
    | sie/Sie 他们/您 | sind      | heißen      | machen        | fragen     | haben        |

+ 不规则动词-换音动词：

    |                 | sein (be) |      |      |      | haben (have) | geben (give) | essen (eat) |
    | --------------- | --------- | ---- | ---- | ---- | ------------ | ------------ | ----------- |
    | ich 我          | bin       |      |      |      | habe         | gebe         | esse        |
    | du 你           | bist      |      |      |      | hast         | gibst        | isst        |
    | er/sie/es 他    | ist       |      |      |      | hat          | gibt         | isst        |
    | wir 我们        | sind      |      |      |      | haben        | geben        | essen       |
    | ihr 你们        | seid      |      |      |      | **habt**     | **gebt**     | **esst**    |
    | sie/Sie 他们/您 | sind      |      |      |      | haben        | geben        | essen       |

+ 不规则动词-变音动词：

    |                 | sein (be) | fahren |      |      | haben (have) | geben (give) | essen (eat) |
    | --------------- | --------- | ------ | ---- | ---- | ------------ | ------------ | ----------- |
    | ich 我          | bin       | fahre  |      |      | habe         | gebe         | esse        |
    | du 你           | bist      | fährst |      |      | hast         | gibst        | isst        |
    | er/sie/es 他    | ist       | fährt  |      |      | hat          | gibt         | isst        |
    | wir 我们        | sind      | fahren |      |      | haben        | geben        | essen       |
