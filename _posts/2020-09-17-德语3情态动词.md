## 语法

1. 助动词，后接动词不定式
2. 独立动词（一般动词）

<br>

## 变格

| 不定式    | mögen        | möchte   |      |      |      |
| --------- | ------------ | -------- | ---- | ---- | ---- |
| 释义      | 喜欢，想，要 | 想要     |      |      |      |
| ich       | mag          | möchte   |      |      |      |
| du        | magst        | möchtest |      |      |      |
| er/sie/es | mag          | möchte   |      |      |      |
| wir       | mögen        | möchten  |      |      |      |
| ihr       | mögt         | möchtet  |      |      |      |
| sie       | mögen        | möchten  |      |      |      |
| Sie       | mögen        | möchten  |      |      |      |
